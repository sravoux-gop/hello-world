# Blind Test App – Spécification initiale

Ce dépôt contient la **spécification fonctionnelle et technique** d'un projet d'application blind test mobile web (Android/iOS via navigateur), hébergée sur Azure.

- Document principal : `docs/blind-test-app-plan.md`

## Fonctionnalités clés visées
- Accès joueur via QR code.
- Saisie nom/prénom.
- Buzzer temps réel.
- Validation/refus côté administrateur avec score +1/-1.
- Classement final décroissant en fin de partie.
